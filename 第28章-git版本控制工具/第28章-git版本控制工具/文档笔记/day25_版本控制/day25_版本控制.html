一、什么叫做版本控制
两个疑问:
    1、什么是版本控制？
    2、为什么要用版本控制工具？
【注】技术源于生活，高于生活。
1、不安全
2、容易丢
3、忘记放哪
4、不方便

代码->钱     电脑->兜

银联卡的特征去解释:
   存在银联卡里的钱    和在自己收的钱
       1、受保护的
       2、受限制的


版本控制工具
                        如何与银联卡对应？
                        1、个人的代码    --    口袋里的钱(放哪随便，丢了难找)
                        2、版本控制工具    --    银联卡中的钱
                        3、版本控制工具里面的客户端    --    银联卡
                        4、版本控制工具里面的服务端    --    银联系统

                        版本控制工具 -- SVN:
                        SVN --    客户端: TortoiseSVN   
                        SVN --    服务端: VisualSVN
            SVN: 集中式管理工具
     github：分布式管理
